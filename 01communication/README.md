## 01communication

### 耗时
6个小时

### 不足
* 服务端和客户端有可共用的代码，可整理
* 传送数据可以加些验证，判断格式是否正确

### 项目搭建和启动说明

- server tcp 服务端

- 安装
```
cd server && npm i
```

- 编译
```
npm run build
```

- 运行
```
npm run dev
```

- client tcp 客户端

- 安装
```
cd client && npm i
```

- 编译
```
npm run build
```

- 运行
```
npm run dev
```
